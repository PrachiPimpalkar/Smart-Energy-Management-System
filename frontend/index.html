📡 IoT Integration Architecture

In a real-world deployment, smart meters and IoT sensors would stream data continuously using MQTT protocol (lightweight, efficient for IoT) and Zigbee (IEEE 802.15.4) for low-power wireless communication. The cloud platform would store readings every 15 minutes, and the ML model would generate predictions for the next 24 hours automatically.

📡 Smart Meters → MQTT Broker → Cloud DB → ML Model → Dashboard
